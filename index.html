Пункт 7. Безусловно null не равняется нулю. При строгом сравнении он не равняется и undefined. При нестрогом - они равны меж собой и ничему другому. Ноль - вполне конкретное числовое значение, которое может быть получено, например как результат вычислений 1-1=0 null - всегда характеризует отсутствие значения. При этом сравнение null >= 0 будет истиной. Потому что null при вычислении сначала преобразуется к числу - ноль. А потом ноль будет сравниваться с нолем.
